🎓 Student Performance Prediction System
📌 Overview

This project is a Machine Learning-based application that predicts a student’s final exam performance based on key factors such as study hours, attendance, previous marks, and sleep patterns. The system uses a Linear Regression model to analyze the relationship between these features and generate accurate predictions.

🚀 Features
Predicts student final marks using Machine Learning
Uses multiple input parameters for better accuracy
Interactive user interface built with Streamlit
Provides real-time predictions
Simple and easy-to-use design
🧠 Technologies Used
Python
Pandas
NumPy
Scikit-learn
Streamlit
Matplotlib
⚙️ How It Works
The dataset is loaded using Pandas
Input features and target variable are defined
Data is split into training and testing sets
A Linear Regression model is trained
The model predicts student performance based on user input
▶️ How to Run the Project
Clone the repository

Install required libraries:

pip install pandas numpy scikit-learn matplotlib streamlit

Run the application:

python -m streamlit run ui.py
Open the browser and use the sliders to input data
